MarkDown

MarkDown语法(兼容HTML语法)

特殊字符自动转换

"<" &lt;
"&" &amp;

标题

Setext形式
    "====="最高阶标题
    "-----"第二阶标题
Atx形式
    '#'一级标题
    '##'二级标题
    '###'三级标题......

列表

无需列表，在文字前加上'-'或'*'或'+'
有序列表，在文字前加'1.''2.''3.'
符号和文字之前加上一个字符的空格

引用

'>'代表添加引用，引用可根据前面加的">"符号个数进行嵌套

插入图片和连接

图片：![](){ImgCap}{/ImgCap}
链接：[链接文字](链接地址"Title")

粗体与斜体

粗体：两个'*'包含
斜体：一个'*'包含

表格

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |

代码

标记一段行内代码：两个"`"将代码包括起来
代码区块：或是缩进四个空格或一个制表符

分割线

"***"
