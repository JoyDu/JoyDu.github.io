Angularjs_01

Posté le 2017-08-23 | In Angularjs 指令

MVC(Angularjs)

视图 文档模型对象
控制器 类，js用方法模拟类
模型数据 被存储在对象中的属性

数据绑定(MVVM)

能自动将Model和View中的数据做同步(能将Model当成唯一可信的数据来源)

Angularjs将数据模型和模板传递到浏览器，生成实时更新的视图，所以无论在model或者view上发生变化 都会立刻反应到对方

依赖注入

用来处理代码的依赖关系

ng-app

指定angularjs应用边界，告诉angularjs应该管理哪一块，也可理解为根元素。HTML文档中默认只有一个ng-app会生效，如果有多个则需要手动执行

angularjs表达式

{{expression}}类似javascript表达式，包含文字、运算符、变量；也可进行计算

ng-init

//初始化angularjs变量
<div ng-init="name='张三'">
    <input type="text" ng-bind="name"/>
</div>

ng-repeat

<ul ng-init="names=['张三','李四','王五']">
    <li ng-repeat="x in names">{{x}}</li>
</ul>

ng-bind

$scope=>view的单项绑定，相当于{{object.xx}}，用于展示数据

ng-model

$scope<=>view双向绑定

数据绑定

<body ng-app>
<input type="text" ng-model="name" value="" placeholder="">
<input type="text" ng-model="name" value="" placeholder="">
{{name}}
<script src="../vendor/angular/angularjs.js"></script>
</body>
//修改第一个文本框的value则第二个文本框和变量name的值都会变化，由此证明angularjs的数据绑定是双向的

自定义指令

<body ng-app="myApp">
<runoob-directive></runoob-directive>
<script>
var app = angular.module("myApp", []);
app.directive("runoobDirective", function() {
    return {
        template : "<h1>自定义指令!</h1>"
    };
});
</script>
</body>
